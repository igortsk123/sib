# Роли и доступ — Tier 1 сводка

## Роли (§8) и право редактировать
- **Владелец/админ организации:** регистрирует организацию, создаёт пользователей/роли, подключает
  ящики, ведёт справочник страховых, видит всё, настраивает экспорт, смотрит аудит-лог.
- **Специалист по ДМС** (основной): видит все письма, ищет, открывает карточку, **правит поля**, связывает
  архив↔пароль, меняет статус, выгружает Excel, комментирует.
- **Врач:** ищет пациента, видит статус согласования/услуги/срок/оригинал; **не правит** финансовые/страховые поля.
- **Регистратура:** ищет (ФИО/ДР/полис/страховая), видит статус/срок/услуги/краткий комментарий; не правит.
- **Старший регистратуры:** + выгрузка Excel, просмотр спорных, комментарии, возможно подтверждение правок.

## Безопасность ПДн/мед.тайна (ADR D4, §16)
- Авторизация + **RBAC**; хэш паролей; **шифрование токенов ящиков**; HTTPS.
- Доступ к оригиналам писем/вложений — по правам; защита от массовой выгрузки без прав.
- **Аудит-лог** всех действий (AuditLog). Бэкапы. Обезличивание/удаление по регламенту.
- Кто оператор данных, где хранятся ПДн, сроки хранения — **решение владельца** (`core/human-decisions.md`).
- Protected-paths и работа с секретами — `.claude/rules/guardrails.md`.
