# Человеческие решения — что спрашивать у владельца

> Принцип (ADR D3): спрашивай владельца **только** по пунктам ниже. Всё остальное (контракты, схемы,
> выбор библиотек, структура кода, тесты) — решай сам, фиксируй ADR в `decisions.md`. Вопрос не из
> списка → сначала реши как обратимую гипотезу и проверь; настоящий тупик → эскалируй и добавь пункт сюда.

## Бизнес / спека (бриф §19)
- **Почтовые ящики:** сколько рабочих; все ли Яндекс или есть Mail.ru/Gmail/Exchange/корп.; доступ по
  IMAP или только веб; будут ли app-password; удалять/помечать письма после обработки; только новые или весь архив.
- **Тестовые письма:** сколько; есть ли по примеру каждого типа (тело/PDF/Word/архив+пароль) и от каждой
  страховой; хранить ли всю цепочку пересылки; есть ли письма с несколькими пациентами/гарантиями.
- **Данные/Excel:** обязательные поля реестра; **утверждённый шаблон Excel** для импорта в систему клиники;
  нужен ли № карты пациента из МИС; распознавать ли диагнозы / суммы-лимиты / коды услуг по прайсу;
  сопоставлять ли услуги с внутренним справочником.
- **Пользователи/роли:** сколько на старте; кто может править распознанное; видит ли врач оригиналы;
  выгружает ли регистратура Excel; нужна ли роль «только просмотр»; нужна ли 2FA.
- **Интеграции:** в какую систему импортируют Excel и её требования (колонки/даты/кодировка/справочники);
  нужен ли API в будущем; уведомления сотрудникам о новых письмах; Telegram/WhatsApp/чат.

## Безопасность / юр. (§16, §19.6) — оператор данных
- ✅ **Хостинг ПДн (2026-06-21): целевой = собственный РФ-сервер мед.клиники** (данные под контролем
  клиники). **Полное логирование разрешено** (ADR D10), в логи не пишем только секреты. Текущий прод —
  интерим на общем VPS (193.160.208.41); перенос на сервер клиники — при выходе из теста.
- Осталось: внутренние требования к ПДн; сроки хранения оригиналов/вложений; шифрование файлов на диске;
  доступ к бэкапам.

## Доступы / деньги
- Прод-хостинг и оплата; доступ к рабочим ящикам клиники; платные лимиты (OCR/LLM) при росте объёма.

## Приоритеты
- Порядок вертикальных срезов и что считать «первым ценным» для демонстрации клинике.
