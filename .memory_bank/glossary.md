# Glossary — термины ДМС и проекта

> Единый словарь: и для кода (идентификаторы), и для UI. Названия страховых → единый справочник (§4).

## Домен
| Термин | Значение |
|--------|----------|
| **ДМС** | Добровольное медицинское страхование. |
| **Гарантийное письмо** | Ответ страховой: подтверждение/отказ/условия оказания мед. услуги пациенту по полису ДМС (срок, лимит, франшиза, перечень услуг). Ядро продукта. |
| **Полис ДМС** | Договор страхования пациента (номер/серия — ключ поиска). |
| **Страхователь** | Организация-работодатель, заключившая ДМС (не сам пациент). |
| **Статус согласования** | согласовано / отказано / частично / требуется уточнение / доп. информация. |
| **Лимит суммы** | Предел покрытия по гарантии. **Франшиза** — часть, оплачиваемая пациентом. |
| **Срок действия** | Период валидности гарантийного письма. |
| **№ обращения / направления** | Идентификатор запроса в страховой. |

## Проект (код)
| Термин | Значение |
|--------|----------|
| **EmailMessage / Attachment / GuaranteeLetter / InsuranceCompany / User / AuditLog** | Сущности БД (`core/data-model.md`). |
| **Двойная пересылка** | клиника→разработчик→тестовый ящик; различать тех. и исходного отправителя (ADR D5). |
| **Ручная очередь** | Записи с низкой уверенностью / неполные — на проверку специалистом ДМС. |
| **Confidence / уверенность** | Оценка надёжности распознанного поля + ссылка на фрагмент-источник. |
| **Срез** | Вертикальная фича UI→БД с тестами (конституция §2). |

## Язык / форматы
- UI: русский. Идентификаторы в коде: English. Даты в UI: ДД.ММ.ГГГГ. Названия страховых — из справочника.
