## Что МОЖНО с осторожностью
- Shared-компоненты/типы — только аддитивно, без изменения рендера/контракта боевых страниц.

## Чеклист перед каждой правкой
1. Задевает файл из «НЕЛЬЗЯ»? → STOP.
2. Может сломать рендер/функционал боевого флоу? → STOP.
3. Может сломать боевой endpoint? → использовать новый, не трогать существующий.
4. Может сломать SEO/маршруты (404/редирект)? → STOP.
